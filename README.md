### Installation

1. Clone the repo
```sh
git clone git@github.com:StephanieFriend/paired_adopt_dont_shop_2001.git
```
2. Open the CLI 
  - Run ```bundle install``` to have all gems needed 
  - Run ```rails db:create && db:migrate && db:seed```
  
3. On the CLI open your local host 
  - ```rails s```
